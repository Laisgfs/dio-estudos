# Digital Innovation One Estudos

Repositório destinado aos desafios, exercícios e projetos desenvolvidos durante as formações da DIO.
O objetivo deste repositório é documentar minha evolução nos estudos de programação, banco de dados e desenvolvimento de software.
Cada desafio possui sua própria documentação (`README.md`), contendo a descrição do projeto, os conceitos praticados e as instruções para execução.

## Estrutura do repositório
Os projetos estão organizados por formação, facilitando a navegação e o acompanhamento da minha evolução durante os estudos.
