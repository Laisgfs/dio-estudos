# Classificador de Nível de Herói - Edição Chapéus de Palha

Este projeto foi desenvolvido como solução para o desafio **Classificador de Nível de Herói**, proposto na formação de **Lógica de Programação** da **Digital Innovation One (DIO)**, ministrada por **Felipe Silva Aguiar**.

## Sobre o projeto

Embora o desafio tenha sido apresentado originalmente em JavaScript, optei por desenvolvê-lo em **Java 21** com o objetivo de reforçar meus estudos na linguagem.

Como forma de personalizar a atividade, adaptei a lógica para o universo do anime **One Piece**. Em vez de utilizar valores fictícios de experiência (XP), o programa utiliza as recompensas oficiais dos integrantes do **Bando do Chapéu de Palha**, representando o nível de cada personagem.

Devido aos valores elevados dessas recompensas, foi utilizado o tipo `long`, permitindo armazenar números na casa dos bilhões sem perda de precisão.

## Objetivo do desafio

O objetivo do desafio é criar um algoritmo capaz de armazenar o nome e a quantidade de experiência (XP) de um herói e, por meio de estruturas condicionais, determinar seu nível conforme faixas de valores previamente definidas.

Nesta adaptação, a lógica original foi mantida, alterando apenas o contexto para o universo de **One Piece**.

## Conceitos praticados

- Variáveis e tipos de dados (`String` e `long`);
- Operadores relacionais e lógicos;
- Estruturas condicionais (`if`, `else if` e `else`);
- Entrada de dados utilizando `Scanner`;
- Formatação de números com `NumberFormat`;
- Organização de código em Java.

- Java 21

- Git

## Como executar

1. Abra o projeto em uma IDE compatível com Java.
2. Navegue até a pasta:

```text
logica-de-programacao/desafio-01-classificador-heroi/src/Main.java
```

3. Execute a classe `Main.java`.
4. Informe o nome de um integrante do Bando do Chapéu de Palha para visualizar sua recompensa correspondente.
